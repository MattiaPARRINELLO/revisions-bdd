## ✅ Comment réviser

1. Ouvre la **cheatsheet** qui correspond à ce que tu veux réviser
2. Va dans `exercices/` et prends un exercice
3. Essaye de le faire **sur papier** sans aide
4. Vérifie avec la correction dans `exercices/corrections/`

## 🎯 Objectifs

- Maîtriser le **style de code** attendu (snake_case, espacement, sections...)
- Maîtriser les **opérations MongoDB** (find, insert, update, delete, sort)
- Savoir enchaîner Python + PyMongo pour un petit programme complet

# Exercice 9 — Écrire des fonctions (style du prof)

Écris les fonctions suivantes **en respectant TOUTES les conventions** du prof :

1. **`get_collection(db, name)`** — retourne `db[name]`

2. **`add_player(db, username)`** — insère `{"username": username, "score": 0, "monney": 100}` dans "players"

3. **`player_exists(db, username)`** — cherche un joueur, retourne True s'il existe, False sinon

4. **`delete_low_scores(db, min_score)`** — supprime les joueurs avec un score < min_score

Chaque fonction doit avoir :
- snake_case
- 2 lignes vides après
- commentaire de section si besoin
- paramètres explicites
